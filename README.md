# Frontend of a Restaurant Menu Page

## Project Objective
The objective of this project is to design and build a responsive and user-friendly
restaurant menu website using HTML, CSS, and JavaScript. The website shows food
items by category and allows users to filter them.

## Project Description
This is a frontend-only restaurant menu website.
Menu items are grouped into categories like Starters, Main Course, Desserts, and Beverages.
JavaScript is used to filter menu items.
The design works on mobile and desktop screens.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Folder Structure
restaurant-menu/
- index.html
- style.css
- script.js

## Features Implemented
- Category-wise menu display
- Filter menu using JavaScript
- Responsive design
- Card layout for food items
